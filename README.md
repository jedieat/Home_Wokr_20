Программа сервер/клиент TCP по Windows
И сервер и клиент реализован в одном cpp serverTCP.cpp.
Надо сначала запустить сервер, потом клиент.
При запуске в меню сначала выбрать 1, потом запустить еще раз, выбрать 2.
   "TCP Server/Client" << endl;
     "1. Server запущен" << endl;
     "2. Client запущен" << endl;
     "Выберите 1 или 2: ";

     После подключения можно обмениваться сообщениями
